    NewWFC is an alternative to AltWFC which discontinued the game Mario Kart Wii. The project is ran by ian47vr and is based off of the DWC script. Please always use RCE protection when connecting to newwfc, and understand you are at your own risk while using the server. Thank you to 2awef for creating most of this webpage and MaritoGuionYo for helping me maintain NewWFC. This server only officially supports Mario Kart Wii but other games such as MKDS and SSBB do work on the server, no game is blocked from connecting to the server but many will not work. You can connect to the server via patched iso/ main.dol (patch for newwfc.xyz), DNS (we support the domains nintendowifi.net, zwei.moe, benfi.ml, and wiimmfi.de), and the connection code. Join the discord for technical support or to search for other players!

    This server has no rules so beware of potential hackers, rulebreakers, and trollers in public lobbies. We will never ban for any reason! Please avoid hacking FTW, it is annoying.
